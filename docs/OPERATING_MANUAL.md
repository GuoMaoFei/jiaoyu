# 智树 (TreeEdu) Agent 操作手册

欢迎使用 **智树 (TreeEdu) Agent**！本手册将引导您了解系统的各项功能，帮助学生实现高效、个性化的自主学习，同时也为家长提供便捷的学情监督。

---

## 1. 快速入门

### 1.1 登录与注册
访问应用首页 `http://localhost:5173/`。
- **学生登录**：输入用户名和密码即可进入学习中心。
- **家长注册**：点击“立即注册”，创建家长账户以查看孩子的学情周报。

![登录页面](file:///C:/Users/茂飞/.gemini/antigravity/brain/53362139-73ef-404b-bc52-75b27f190f50/login_page_1773151675220.png)

### 1.2 仪表盘概览
登录后，您将看到“今日任务”仪表盘。
- **今日任务**：展示 AI 为您定制的学习计划，包括新知学习、微测和诊断。
- **学习统计**：实时显示今日学习时长和获得的知识点总数。
- **计划达成率**：通过环形进度条直观了解本周计划进度。

## 2. 核心功能操作

### 2.1 统一书架与教材管理
在侧边栏点击“统一书架”，管理您的学习资源。
- **添加教材**：点击右上角“添加教材”，可选择官方预置教材或上传自定义 PDF 教材。
- **教材激活**：点击教材卡片上的“加入我的书架”即可激活，系统将自动生成对应的知识树。

![统一书架](file:///C:/Users/茂飞/.gemini/antigravity/brain/53362139-73ef-404b-bc52-75b27f190f50/bookshelf_1773151577524.png)

### 2.2 知识书林 (可视化知识树)
点击“知识书林”，查看课程的结构化图谱。
- **颜色标识**：
  - <span style="color:green">●</span> **掌握**：该知识点已通过测试，掌握程度高。
  - <span style="color:orange">●</span> **巩固**：需要进一步练习。
  - <span style="color:red">●</span> **薄弱**：错误率较高，需重点关注。
  - <span style="color:grey">●</span> **锁定**：尚未开始学习。
- **交互**：点击树节点可快速跳转至对应章节的学习或微测。

![知识书林](file:///C:/Users/茂飞/.gemini/antigravity/brain/53362139-73ef-404b-bc52-75b27f190f50/knowledge_forest_1773151567740.png)

### 2.3 AI 伴读与学习舱
在今日任务中点击“开始学习”，进入智能辅导环境。
- **苏格拉底式对话**：AI 助手“伴读神仙”不会直接给答案，而是通过引导式提问帮助您理解核心概念。
- **多模态提问**：支持文字输入，Pad/手机端支持拍题提问。
- **进度追踪**：系统自动记录对话过程，动态更新您的学情。

![AI 伴读](file:///C:/Users/茂飞/.gemini/antigravity/brain/53362139-73ef-404b-bc52-75b27f190f50/ai_tutor_chat_1773151630881.png)

## 3. 评估与报告

### 3.1 测验系统
- **入学诊断**：新教材激活后的全量摸底，帮助 AI 初始定位您的水平。
- **知识点微测**：学习完一个小模块后的即时验证。
- **阶段大考**：基于已选教材和解锁进度的综合性模拟测验。

### 3.2 学情周报
点击侧边栏“学情周报”，查看 AI 生成的深度分析。
- **柔性报告**：采用鼓励式话术，总结一周的学习亮点和待改进项。
- **情绪状态**：监控交互中的情绪波动，提供心理关怀建议。

## 4. 常见问题 (FAQ)

**Q: 上传的 PDF 教材解析需要多久？**
A: 解析速度取决于文件大小，通常 1-3 分钟内完成。解析完成后，您可以在书架中看到生成的知识树。

**Q: 为什么我无法跳转到下一节学习？**
A: 请确认当前节点的“微测”是否已通过。系统遵循“最近发展区”理论，确保掌握基础后再进阶。

**Q: 家长如何查看孩子的进度？**
A: 家长只需使用注册的家长账号登录，即可在首页或消息中心看到专属的学情周报。
